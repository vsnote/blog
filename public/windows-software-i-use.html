我用的 Windows 软件

March 3, 2013

虽然我自用的笔记本电脑装的是 Ubuntu 系统，但工作中还是在使用 Windows 系统：

  1. 浏览器 – firefox + vimperator
  2. 文本编辑器 – Vim
  3. 剪贴板 – ditto
  4. 自动化 – autoHotKey
  5. 搜索 – everything，仅支持 NTFS 文件系统
  6. 快速启动 – find and run robot
  7. 截屏软件 – screenshot captor
  8. 代理 – Privoxy，用于过滤页面广告
  9. 解压软件 – 7-zip
  10. 搜索 – Listary
  11. 代码编辑器 – Sublime Text 2
