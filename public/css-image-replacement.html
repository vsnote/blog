CSS 隐藏文本

August 8, 2013

一个元素，需要背景图片替代文字，但又想在 HTML 中保留文字，最常见的解决方法是：

p{text-indent:-9999px;}

但据说这种办法可能被搜索引擎判断为作弊行为。

可我是前端开发，不是搜索引擎优化人员，这种后果暂时就不去想了。

HTML5Boilerplate 里隐藏文本用的方法如下：

/*
* Image replacement
*/

.ir {
    background-color: transparent;
    border: 0;
    overflow: hidden;
    /* IE 6/7 fallback */
    *text-indent: -9999px;
}

.ir:before {
    content: "";
    display: block;
    width: 0;
    height: 150%;
}

这个方法使用 :before 伪类，在元素前插入空内容块，并设置其高度为元素高度的150%，这样，真实文本超出包含块高度 – 因为设置 overflow:hidden;，也隐藏了文本。

* 是针对 IE6、7浏览器的，可以改造如下：

.ir {
    background-color: transparent;
    border: 0;
    overflow: hidden;
}
.lt-ie8 .ir{
    text-indent:-9999px;
}
.ir:before {
    content: "";
    display: block;
    width: 0;
    height: 150%;
}

这样就符合 CSS 规范了。
