openSUSE 下快速切换打开的窗口

November 11, 2013

我一直认为，Alt + Tab 这样的切换运行任务的方式非常低效。这也是我在 Windows 系统下使用 iSwitchw 脚本的原因。

openSUSE 有类似功能，不过颇隐蔽。

按 Alt - F1，打开桌面左下的主菜单，然后点击 Configure Desktop(配置桌面)，进入界面后，打开 Shortcuts and Gestures，定位到 Global Keyboard Shortcuts，KDE component 下拉菜单中选择 Run Command Interface，然后配置下方的 「Runner “Windows” only」：

我的习惯，是使用 Win + W 键。

调出小窗口后，输入要定位的窗口标题，然后回车即可。

补充

KDE 下其实还有一个预览当前所有打开窗口的方法，只要把光标移动到桌面左上角，或者按Ctrl-F9(预览当前桌面所有打开窗口)、Ctrl-F10(预览所有桌面的所有打开窗口)。之后在预览里输入标题信息即可自动过滤。
