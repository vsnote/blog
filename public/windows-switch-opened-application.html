Windows 系统快速切换运行中的任务窗口

April 4, 2013

如果用用 Windows XP 系统，我可以按 Alt+Tab 切换运行中的任务窗口，又或者 Alt+Esc。

Windows 7 中增加快捷键 WIN+Tab，视觉上有立体效果，见下图（图片来自 Microsoft 网站），

但我之所以用它，只是因为好看。

总体来说，它们的定位效率都太低。

我的电脑上安装 Enlightenment 17 桌面系统，它有个功能，通过打开窗口的标题来快速切换任务。比如我系统里运行着 firefox、Google Chrome、Sublime Text 2、Tmux，则启动该功能后，键入 fi 后按 Enter 就可以切换到 firefox 窗口。因为知道自己要打开哪个窗口，所以大概地知道窗口标题不是件难事。

Windows 下要增加这种功能，可以借助 AutoHotkey 脚本。

  1. 首先安装 AutoHotkey
  2. 然后新建一个 .ahk 文件
  3. 将脚本内容拷入，保存
  4. 运行 ahk 文件。

脚本默认使用 CapsLock 键来触发，触发后桌面会弹出一个窗口，键入窗口标题就可以快速切换，如下图：

但我们可以根据自己的习惯配置触发键，比如我更倾向使用 Win + Space，因为方便按到。在脚本中查找 CapsLock::，把它替换成 #Space:: – # 在 AutoHotkey 里表示 Win 键。
